Описание задачи
Нужно написать сервис, позволяющий определять футболиста по фотографии. Взаимодействие с сервисом должно осуществляться с помощью api (см python fastapi).


Под капотом должен работать обученный классификатор (произвольная картинка -> лейбл “Команда”+”Номер футболиста” в соответствии с images_labelling.csv).


Описание данных
Данные представляют собой вырезанные bbox'ы футболистов на поле. Лейблы классов можно найти в images_labelling.csv. 
Разрешается (даже поощряется) использовать любые внешние данные, модели в угоду лучшему(может более изящному) решению.
